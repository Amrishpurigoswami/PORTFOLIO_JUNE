# Portfolio — Amrish Puri Goswami

A single-page portfolio site. No frameworks, no build step — just `index.html`.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `portfolio` or `Amrishpurigoswami.github.io` for a root-domain site).
2. Upload `index.html` to the repo (drag-and-drop on github.com works fine, or use git).
3. Go to **Settings → Pages**.
4. Under **Source**, select the `main` branch and `/ (root)` folder, then **Save**.
5. Wait ~1 minute, then visit the URL GitHub gives you (e.g. `https://amrishpurigoswami.github.io/portfolio/`).

## Customize

- Update the LinkedIn link in the Contact section (currently a placeholder `#`).
- Swap project GitHub links for the exact repo URLs once you've pushed each project.
- Edit colors via the CSS variables at the top of `<style>` (`--cyan`, `--amber`, `--violet`).
